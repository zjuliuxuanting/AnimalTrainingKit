③ 背景建模

④ 标尺校准

在画面上点击两点画一条线段，输入该线段对应的真实长度（如10厘米），系统自动计算像素与实际距离的比例。

点击画面上两点，画一条已知长度的线段

⑤ 绘制检测区域

在画面上点击顶点绘制多边形区域。双击最后一个顶点闭合多边形。

⑥ 区域事件定义

为每个检测区域命名，配置事件类型。动物进入/离开/累计N次/停留X秒时会产生触发信号，可在流程编辑器中引用。

⑦ 开始检测

确认所有区域和事件配置无误后，开始摄像头检测。检测事件将写入当前实验的数据记录。

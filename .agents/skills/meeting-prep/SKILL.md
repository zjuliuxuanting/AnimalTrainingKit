# Meeting Prep — 会议准备与跟踪

## 一、外包对接会议准备

### 会前准备清单

- [ ] PRD 已定稿（训练员视角，无技术细节约束）
- [ ] 项目任务书已准备（含里程碑和付款建议）
- [ ] 验收清单已更新
- [ ] 会议议程已确定

### 会议议程模板

**时长建议：1.5 小时**

| 环节 | 时长 | 内容 |
|------|------|------|
| 甲方讲场景 | 10min | 为什么做这个、现在遇到什么问题、期望的最终效果 |
| 甲方带读 PRD | 15min | 逐节过 PRD，重点讲"要什么" |
| 外包方提问 | 15min | 外包方对需求提问，甲方澄清 |
| 技术方案讨论 | 20min | 外包方初步判断可行性，提方案方向 |
| 合作框架 | 15min | 过任务书：交付物、里程碑、付款 |
| 下一步计划 | 5min | 确定方案文档提交时间、下次沟通时间 |

### 会后跟进

- 会议纪要 24 小时内发出
- 记录外包方的疑问点和甲方的回答
- 更新决策记录（如果有新决策）
- 看板上更新状态

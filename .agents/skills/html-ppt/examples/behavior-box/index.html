// context

行为学实验的痛点：
数据采集靠手记，流程执行靠人盯。

传统手动方式

实验人员手动计时、手工记录数据。费时费力，数据一致性差，难以复现实验结果。

❌ 不可靠

商用方案

Med Associates、Noldus 等商业系统功能全但价格昂贵（数万～数十万），封闭不可定制。

💰 太贵

行为学训练盒

开源、低成本（目标 140 元/台）、模块化。摄像头 + ESP32 传感器 + 可视化流程编排。

✅ 开源可定制

// highlights

核心亮点

1 · 统一信号抽象

摄像头、传感器、定时器、模拟信号归一化为 SignalEvent，支持任意组合。

2 · 可视化流程编排

拖拽式节点编辑器，10 种节点类型，实验人员无需编程即可设计实验。

3 · 零硬件启动

Mock 信号源模式，无需任何硬件即可验证完整实验流程，大幅降低开发门槛。

4 · TDD + 严格角色边界

PM / 软工 / 测试员三角色协作，红线清晰。红-绿-重构循环保障质量。

5 · 开源模块化

完整 Python 源码，纯前端无框架，协议模块化，易于定制和二次开发。

6 · 7 步摄像头向导

从摄像头选择到追踪目标定义，完整视觉引导流程，生物学背景也可上手。

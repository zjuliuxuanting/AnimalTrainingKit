Pain Points

13 Node Types

拖拽连线 · 零代码 · 就像画流程图

📡

信号输入 (3)

TRIGGER — 啄击/摄像头运动
SNIFFER — 旁路监听
START — 流程起点

🔄

流程控制 (4)

DELAY — 延时等待
LOOP — 循环+超时退出
CONDITION — 真/假分支
END — 流程终点

🧮

逻辑运算 (4)

AND — 全部满足触发
OR — 任一满足触发
NOT — 信号反转
FORK — 并行分叉

⚡

动作执行 (2)

EXECUTE — 出粮/脉冲/开关
RECORD — 日志埋点

🔗 完整链路：START → TRIGGER(啄击) → AND → EXECUTE(出粮) → RECORD → LOOP → ... → END
